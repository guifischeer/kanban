# Kanban - Autenticação, Sessão e Integração com API Pública
Arquivos gerados automaticamente para o seu projeto Kanban.

## Instruções rápidas
1. Extraia os arquivos em `htdocs/kanban` (XAMPP) ou diretório equivalente.
2. Crie o banco de dados `kanban` e execute `database.sql`.
3. Ajuste `db_connect.php` com seu usuário e senha do MySQL.
4. Acesse `http://localhost/kanban/index.php`.

## Conteúdo
- arquivos PHP principais: index.php, cadastro.php, menu.php, logout.php
- pasta `public/` com endpoints e operações de tarefas
- pasta `api/` com exemplo de integração ViaCEP
- `styles/style.css`
- `database.sql` com o schema
